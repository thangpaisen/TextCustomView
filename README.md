## Installation

Using npm:

```
npm i --save rn-text-custom
```

Using yarn:

```
yarn add rn-text-custom
```

## Usage

```jsx
import {TextCustomView} from 'rn-text-custom';

<TextCustomView
  bold
  fontSize={20}
  color="red"
  style={{
    padding: 20,
  }}>
  Hello World!
</TextCustomView>;
```
